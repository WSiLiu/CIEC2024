This code is written with Matlab R2023B, derived from the source code of 'Kappel D, Nessler B, Maass W (2014) STDP Installs in Winner-Take-All Circuits an Online Approximation to Hidden Markov Model Learning. PLoS Comput Biol 10(3): e1003511.'
This code constructs a feedforward k-WTA spiking neural network which implements Bayesian inference and simulates the basic phenomenon of simultaneous brightness contrast, with the main references as 'Liu, W., Liu, X. The effects of distractors on brightness perception based on a spiking network. Sci Rep 13, 1517 (2023).'
